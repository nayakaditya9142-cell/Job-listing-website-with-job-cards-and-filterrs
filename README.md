# EmployHub - Job Listing Platform

A modern, responsive job listing website built with Vanilla HTML, CSS, and JavaScript. This project features dynamic job cards, real-time filtering, job saving, and a detailed modal view.

## Features

- **Dynamic Job Cards**: All jobs are rendered dynamically from a data object.
- **Search & Filters**: 
  - Search by job title or keyword.
  - Filter by Location (including Remote).
  - Filter by Category (IT, Marketing, Design).
  - Filter by Experience Level (Fresher, Mid-level, Senior).
- **Interactive Details**: Click "View Details" to see full job descriptions and requirements in a sleek modal.
- **Save Jobs**: Bookmark jobs for later with a "Save" button that persists in local storage.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Premium UI**: Uses a clean, modern aesthetic with glassmorphism effects and smooth transitions.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Layout (Flexbox & Grid), animations, and responsive design.
- **JavaScript (ES6)**: State management, filtering logic, and DOM manipulation.
- **Lucide Icons**: Premium vector icons.
- **Google Fonts**: Inter (Body) and Outfit (Heading) fonts.

## Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```
2. **Open the project**:
   Simply open `index.html` in any modern web browser.
3. **Filtering**:
   Use the search bar or the dropdown filters on the left to find specific jobs.
4. **Saving Jobs**:
   Click the bookmark icon on any job card to save it.

## Project Structure

- `index.html`: Main landing page structure.
- `style.css`: All styles including responsive media queries.
- `script.js`: Core logic for filtering and interactions.
- `jobs.js`: Mock data containing job listings.
